### selectメソッドとrejectメソッドの違い

- selectメソッド：
調べたブロックが真であった場合、その要素を返す。
```ruby
(1..10).select {|i| i % 2 == 0 }
=> [2, 4, 6, 8, 10]
```
```ruby
(3, 5, 7, 9).select {|i| i % 2 == 0 }
=> []
```
- rejectメソッド：
調べたブロックの中で、偽であった要素を返す。
```ruby
(1..10).reject {|i| i % 2 == 0 }
=> [1, 3, 5, 7, 9]
```
```ruby
[2, 4, 6].reject { |i| i % 2 == 0 }
=> []
```
